# 「文案实验室」Pro：引导式创作与全场景改写设计方案

本方案在保持原有“苹果风格改写”核心的基础上，引入**“智能场景引导 (Scenario Wizard)”**，通过特定字段输入辅助 AI 生成更高质量、更具针对性的文案。

## 1. UI 架构设计 (Apple 风格升级)
*   **首页：双模式中心**：
    *   **模式一：自由改写**（保持当前 UI）。
    *   **模式二：灵感向导**（新增）：以卡片流展示预设场景，如「哄女友专用」、「职场高光」、「长辈马屁」等。
*   **场景引导弹窗/页面 (Wizard Form)**：
    *   点击场景后，从底部弹出半屏浮层或跳转至简洁的填单页。
    *   **动态字段**：根据场景显示 2-3 个核心问题。例如：
        *   「哄女友」：生气的缘由？（如：迟到、没回消息）
        *   「年会发言」：你的职位？（如：销售主管）；发言背景？（如：庆功宴）
    *   **风格联动**：在引导单底部，用户依然可以选择「王家卫风」、「甄嬛传风」等，让同一场景产生截然不同的趣味效果。

## 2. 核心功能场景与引导路径设计
我为你设计了 6 个核心场景的引导逻辑：

1.  **哄女朋友 (The Peacekeeper)**：
    *   *引导字段*：生气原因、你的认错态度（诚恳/求饶/幽默）。
    *   *推荐风格*：霸道总裁、甄嬛传（“本宫知错了”）、琼瑶风。
2.  **年会演讲 (The Spotlight)**：
    *   *引导字段*：职位、公司主要成就、想感谢的人。
    *   *推荐风格*：小米风（参数感）、苹果风（高级感）、知乎体。
3.  **朋友圈创作 (Moments Pro)**：
    *   *引导字段*：当前心情、所在地点、想表达的核心意思。
    *   *推荐风格*：小红书风、王家卫、废话文学。
4.  **餐厅好评 (Gourmet Review)**：
    *   *引导字段*：菜系、最推荐的菜名、服务亮点。
    *   *推荐风格*：小红书风（带 Emoji）、鲁迅风格（犀利好评）。
5.  **夸夸长辈/丈母娘 (The Respectful)**：
    *   *引导字段*：长辈身份、对方做的让你感动的事（如：做了一桌好菜、带孩子辛苦）。
    *   *推荐风格*：甄嬛传（得体）、鲁迅风格（深沉感人）。
6.  **同辈互夸 (Peer Compliment)**：
    *   *引导字段*：对方的优点（如：审美高、效率快）、具体事例。
    *   *推荐风格*：发疯文学（极致赞美）、知乎体。

## 3. 交互细节与体验优化
*   **智能拼接 Prompt**：系统会将用户的表单输入（如：职位+成就+感谢人）自动转化为一段逻辑连贯的“原始文案”，再交给选定的“风格引擎”进行二次改写。
*   **可视化表单**：使用 Apple 风格的圆角输入框，配合温和的占位提示词，让填写过程像聊天一样自然。
*   **结果对比**：支持在生成结果后，直接切换下方风格，无需重新填写表单。

## 4. 实施阶段计划
1.  **WXML 扩展**：新增底部抽屉组件 (ActionSheet) 用于承载引导表单。
2.  **JS 逻辑重构**：编写 `scenarioManager` 模块，管理不同场景的字段定义与 Prompt 拼接规则。
3.  **样式统一**：确保引导表单的 UI 与主页面保持一致的高级感。
