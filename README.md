# 文案实验室 (TextStylized) 🧪

「文案实验室」是一款基于微信小程序开发的文案风格化工具。它采用 Apple Premium 设计语言，结合先进的大模型能力，旨在帮助用户将平庸的文字转化为极具风格张力的文案。

## ✨ 核心特性

- **🍎 Apple 级视觉设计**：全界面遵循 Apple 设计规范，采用卡片化布局、精致的阴影效果及丝滑的交互动画。
- **🎭 多重风格矩阵**：内置 16 种极具张力的文案风格，包括王家卫、苹果文案、甄嬛传、鲁迅、互联网黑话、古风诗经、浪漫英雄主义、网易云 emo 等。
- **🪄 灵感向导模式**：针对特定场景（哄女友、年会发言、过年祝福等）提供结构化引导，并智能建议匹配风格。
- **🚀 深度能力集成**：通过微信云开发安全调用 AI 接口，实现深度的语义理解与风格转换。
- **📱 Skyline 渲染支持**：深度优化微信最新 Skyline 渲染引擎，提供更接近原生 App 的性能与交互质感。
- **🛡️ 运营管控与安全**：
  - **内容风控**：集成微信官方 `msgSecCheck` 接口，确保生成内容合法合规。
  - **使用限制**：内置每日 3 次免费额度限制，通过云数据库进行实时计数。
  - **增长引导**：额度用尽后自动引导联系管理员解锁，预留商业化接口。

## 🛠️ 技术栈

- **小程序框架**: 原生微信小程序 (WXML/WXSS/JS)
- **渲染引擎**: Skyline (支持 Simple 模式布局)
- **后端支持**: 微信云开发 (Cloud Development)
- **云数据库**: 用于存储用户每日使用配额
- **AI 引擎**: DeepSeek-Chat (通过云函数中转调用)
- **工具库**: Axios (云端请求), Taptic Engine (触感反馈)

## 🚀 快速开始

### 1. 克隆与环境配置
- 在微信开发者工具中导入本项目。
- 开通微信云开发环境。

### 2. 数据库配置
- 在云开发控制台中新建集合：**`user_usage`**。
- 设置权限为「所有用户可读，仅创建者可读写」。

### 3. 云函数部署
- 找到 `cloudfunctions/deepseekService` 文件夹。
- 右键点击并选择「上传并部署：所有文件（云端安装依赖）」。

### 4. API Key 设置
- 进入云开发控制台 -> 云函数 -> `deepseekService` -> 配置。
- 添加环境变量 `DEEPSEEK_API_KEY`，填入你的 DeepSeek API Key。
- 将执行超时时间建议设置为 `60s`。

### 5. 环境初始化
- 在 `app.js` 中确保 `wx.cloud.init` 的 `env` ID 与你的云环境一致。
